# Multicampus CloudMSA 5조 Client
CloudMSA 5조 클라이언트 레포지터리 입니다.
## 사용된 기술
- [ ] React
- [ ] Mobx
- [ ] html,css,javascript
- [ ] Material-ui
## 클라이언트 구동 방법
### git clone
```bash
$ git clone https://github.com/heoseongh/itple-client.git
```
### yarn install
```bash
$ yarn install
```
### yarn start
```bash
$ yarn start
```
